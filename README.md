# OCR Text Extraction Tool

## Overview

A Python OCR application that extracts text from images using Tesseract OCR and Pillow.

## Features

- Extract text from images
- Supports PNG, JPG and JPEG files
- Simple command-line interface
- Error handling
- Easy integration with document-processing workflows

## Technologies Used

- Python
- Tesseract OCR
- Pillow
